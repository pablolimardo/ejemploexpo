Manual de Usuario

Este manual te guiará para poner en marcha y operar el Sistema de Riego Inteligente.

1. Componentes del Sistema

Nuestro prototipo consta de las siguientes partes:

  1. Caja de Control: Contiene el microcontrolador ESP32, que es el cerebro del sistema.
  2. Sensor de Humedad: Es la varilla que se inserta en la tierra para medir cuánta agua hay.
  3. Bomba de Agua: Pequeña bomba sumergible que se coloca en el depósito de agua.
  4. Dashboard Web: Página accesible desde cualquier dispositivo en la red para ver los datos en tiempo real.

2. Puesta en Marcha

  1. Colocá la manguera de la bomba en la maceta o sector a regar. Sumergí la bomba en un recipiente con agua.
  2. Insertá el sensor de humedad en la tierra de la maceta, asegurándote de que tenga buen contacto.
  3. Conectá el sistema a una fuente de alimentación USB de 5V (como un cargador de celular).
  4. El sistema se iniciará automáticamente y se conectará a la red WiFi configurada. Un LED en la caja parpadeará para indicar que está funcionando.

3. Monitoreo del Dashboard

Para ver el estado del sistema, abrí un navegador web y andá a la dirección IP del dispositivo (ej: 192.168.1.105). Verás lo siguiente:

  • Nivel de Humedad: Un medidor o un gráfico que muestra el porcentaje actual de humedad en la tierra.
  • Estado de la Bomba: Un indicador que te dice si la bomba está ENCENDIDA o APAGADA.
  • Umbral de Riego: El valor de humedad por debajo del cual el sistema activará el riego automáticamente.

4. Solución de Problemas Frecuentes

  • "La bomba no se activa": Verificá que el nivel de agua en el depósito sea suficiente y que la humedad en la tierra esté por debajo del umbral configurado.
  • "No veo datos en el dashboard": Asegurate de que el dispositivo esté conectado a la misma red WiFi que tu computadora o celular. Reiniciá el sistema si es necesario.
