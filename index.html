Sistema de Riego Inteligente para Invernaderos Patagónicos

El Problema

En El Calafate, el clima presenta un gran desafío para la producción de hortalizas. Los invernaderos son una solución, pero requieren un uso muy cuidadoso del agua, un recurso que escasea, sobre todo en verano. Un riego manual ineficiente puede desperdiciar agua o, por el contrario, no ser suficiente para las plantas.

Nuestra Solución

Desarrollamos un prototipo de sistema de riego automatizado que utiliza la tecnología para optimizar el uso del agua. El sistema monitorea constantemente la humedad de la tierra y activa una bomba de riego solo cuando es estrictamente necesario, asegurando las condiciones ideales para el crecimiento de las plantas con el mínimo consumo de agua posible.

Tecnologías Utilizadas

  • Hardware: Un microcontrolador ESP32 como cerebro, un sensor de humedad de suelo capacitivo para medir las condiciones de la tierra y una mini bomba de agua de 5V.
  • Software: El control fue programado en el IDE de Arduino (C++). Los datos se envían a través del protocolo MQTT a un dashboard web.
  • Visualización: Creamos una página web simple con HTML, CSS y JavaScript que muestra en tiempo real la humedad del suelo y el estado de la bomba.

Impacto del Proyecto

Este proyecto no solo demuestra una solución práctica a un problema local, sino que también nos permite aplicar los conocimientos de programación, electrónica y conectividad que aprendimos durante el año. Es un pequeño paso para convertirnos en los futuros técnicos que nuestra comunidad necesita.
